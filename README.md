# xst-task-mngt

To install dependencies:

```bash
bun install
```

To start detabase:
```bash
docker compose up -d
```

To run Frontend:
```bash
cd xst-task-mngt/apps/web
bun run dev
```

To run Backend:

```bash
cd xst-task-mngt/apps/api
bun run dev
```

This project was created using `bun init` in bun v1.2.23. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
